## Getting Started

1. Clone Repository

```
git clone https://github.com/themewagon/berry-vue.git
```

2. Install Dependencies

```
npm i
```

3. Run the development server:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```
